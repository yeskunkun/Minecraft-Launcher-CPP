# Minecraft Launcher
<img width="1280" height="800" alt="image" src="https://github.com/user-attachments/assets/9c9b1cde-bf41-4c9b-b755-816be33fb75c" />
C++ WinAPI制作的Minecraft: Java Edition 离线启动器！
用YesUI作为UI框架制作，支持自动下载Java和Minecraft！
支持 Microsoft Windows 10-11 系统。

## 此启动器的亮点？
* 支持自动下载 Minecraft 和解压 Native，所有正式版**全部支持**，快照版的远古版本可能会崩溃。
* 支持自动下载并安装 Java，全程静默安装（UAC除外）。
* 支持根据版本自动匹配对应 Java。
* 支持**Mica效果，深色模式、浅色模式**跟随系统切换。
* 全程使用 **C++** WinAPI 打造，领先于所有启动器项目。

> 此项目还有一些 UI 层面的 Bug 需要修复，其它均完善，能够正常下载 Java、启动 Minecraft。
